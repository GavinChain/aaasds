# vitepress-template

::: warning
注意测试时，首次编译会报错误，请多刷新几次正常。
:::

- [在线帮助文档](https://qve.gitee.io/docs/plugin/)

#### 介绍

vitepress 文档模板，支持站内搜索,markdown 中运行 vue3 组件，与画的时序图，图表组件

#### 安装教程

```sh
# 本地新建目录 拉取模板
git clone https://gitee.com/qve/vitepress-template.git

# 安装

yarn

## 测试

yarn docs:dev

## 打包

yarn docs:build
```
